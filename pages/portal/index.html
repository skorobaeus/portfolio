Portal GKH

Portal GKH (Utility Services Portal) is a web-service which allows citizens to do all the communication with utility services online, starting with payments, ending with calling a plumber.

Tasks: design the user interface, draw vector icons, code e-mail

Tools: Adobe Photoshop, Adobe Illustator

2016 — 2017

Utility Services Portal consists mostly of tables because it's made for exchanging data between citizens and utility services.

I was invited to this project to create lacking UI elements for the main page, such as a register form...

...or icon set for description section.

I designed inner pages relying on prototypes done be product manager and color palette made by previous designers.

Here you can see a "Desktop" page. It is the first page a user sees after logging in. This page provides a quick access to the most popular functions.

Payment page, besides a payment form, had to include a billing history. Wide table defined the design of entire page, including the form. That's why summ and commission info are placed on the sides.

I had to make tables as readable as possible, because it was most demanded type of content.

I also had to provide an option of showing commentaries to table vales. And these commentaries could also be tables! I decided to not use pop-op windows, and to show additional info inside the table itself.

I would design the next page differently these days. The text field better be narrower, so it's easier to read the text inside it. In the message history I'd prioritize the text of the messages, not date or type.

Utility Services Portal was still in development when I left the company.

Портал ЖКХ

Портал ЖКХ это сервис для жителей моего города, который позволяет всё общение со службами ЖКХ вести онлайн, начиная от подачи показаний счётчиков и заканчивая вызовом мастера.

Задачи: разработать дизайн страниц личного кабинета, нарисовать иконки, сверстать почтовую рассылку.

Инструменты: Adobe Photoshop, Adobe Illustator

2016 — 2017

Портал ЖКХ почти полностью состоит из форм и таблиц, так как он предназначен для обмена большими массивами данных между жителями и службами ЖКХ.

Моя работа над проектом началась с разработки недостающих элементов для главной страницы. Например, подробной формы регистрации.

Или набора иконок для блока с описанием возможностей Портала.

Максимум возможностей доступен в личном кабинете пользователя. Я разработала дизайн страниц личного кабинета, опираясь на прототип менеджера проекта и на определённую предыдущими дизайнерами палитру цветов.

«Рабочий стол» это стартовая страница при входе в личный кабинет. Он предоставляет быстрый доступ к услугам, за которыми жители обращаются чаще всего.

На странице оплаты, помимо самой формы оплаты, требовалось так же представить историю платежей в табличном виде. Таблица, тянущаяся вширь, определила и дизайн формы оплаты, поэтому я разместила информацию о сумме к оплате и комиссии слева и справа от формы.

Большая часть контента Портала это таблицы, поэтому было важно сделать их максимально читаемыми.

Также нужна была возможность показать комментарии к значениям в ячейках. И эти комментарии тоже могли иметь табличную форму! Я решила отказаться от всплывающих окон и выводить детализацию прямо в таблице, между строк.

На странице обращений сегодня некоторые элементы я сделала бы по-другому. Например, уменьшила бы ширину поля для ввода, чтобы текст легче читался. В истории обращений первым вывела бы текст сообщения, чтобы было легче найти нужное, т.к. пользователи скорее узнают обращение по первым словам, чем по способу подачи.

Портал ЖКХ был запущен в новом дизайне после моего ухода из компании.
